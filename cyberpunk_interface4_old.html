About iNTRUDER v1.5

This dashboard provides a sleek cyberpunk interface to coordinate your WiFi pentesting tasks. Use the navigation buttons below to access each function:

Features

Monitor Mode

  • One-click activation via /api/monitor/start
  • Uses default or specified interface
  • Supports real-time logs and status

Network Scanning

  • Adaptive scanning via /api/scan/start
  • JSON-formatted results
  • Configurable duration and filters

Handshake Capture

  • Initiate capture with /api/handshake/start
  • Capture by SSID or BSSID
  • Auto-channel hopping support

Deauth Attack

  • Trigger with /api/deauth/start
  • Target BSSID and optional client MAC
  • Customizable packet count

WPA Cracking

  • Start with /api/crack/start
  • Specify handshake file and wordlist
  • Monitor progress in real time

Reporting & Utilities

  • Generate JSON/Markdown reports via /api/reporting/generate
  • View centralized event logs
  • Plugin placeholders: Rogue AP, MITM, WPS
